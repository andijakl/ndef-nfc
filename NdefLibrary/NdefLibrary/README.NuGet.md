# NDEF Library for Proximity APIs / NFC

This package provides classes to parse and compose NDEF (NFC Data Exchange Format) records and messages in .NET.

Highlights:
- Create / parse common record types (URI, Text, Smart Poster, vCard, iCalendar, Bluetooth handover, etc.)
- High-level convenience APIs over raw NDEF byte handling
- Cross-platform (netstandard2.1)

Changes in 5.0.0:
- Target netstandard2.1 (was netstandard1.4)
- Updated NuGet metadata (embedded icon, license file, readme)
- Internal cleanup for modern SDK style
